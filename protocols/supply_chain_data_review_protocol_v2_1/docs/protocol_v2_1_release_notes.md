# Supply Chain Data Review Protocol v2.1 release notes

This release consolidates the late-May revisions made after testing the integrated v2.0 package.

## Included updates

1. **Figure contract finalized**
   - Only canonical SVG output filenames are emitted:
     - `DCF_PRISMA_S_Figure_1_jurisdictional_terms.svg`
     - `DCF_PRISMA_S_Figure_2_supply_chain_terms.svg`
     - `DCF_PRISMA_S_Figure_3_farm_level_terms.svg`
   - Figures use `reports_referencing` as the measure.
   - Charts show only the x and y axes (no bounding frame).
   - Titles are unnumbered.
   - Figure 1 excludes AOI terms and is limited to jurisdictional / landscape terms.

2. **Workbook usability refinement**
   - The example/reference workbook was simplified toward the v1.3 tabular style.
   - It is intended to better distinguish generated outputs from canonical starting metadata.

3. **APA citation updates**
   - Added explicit APA references for the two newly uploaded corpus additions:
     - RTRS draft standard
     - ECF EUDR guidance document

4. **Reference outputs added**
   - The package now includes a v2.1 reference output bundle with the simplified workbook,
     the corrected SVG figures, and a small APA-update CSV.
